About MiRANA

This tool is intended to help visualise Mendelian Randomisation (MR) estimates, in particular where multiple estimates interact with eachother (e.g., Multivariate, Two-step & Network MR).

It has been developed by the Dynamic Genetics lab group at the MRC Integrative Epidemiology Units (University of Bristol).

Develop a graph in just a few steps:

  1. Store MR estimates in a table (.csv file)
  2. Upload the file to the tool
  3. Click generate graph
  4. Optionally, the nodes will stick so you can re-arrange the graph as you wish
This is open-source project and the git-hub repository is located here.
